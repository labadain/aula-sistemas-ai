Varanda Silabus

Progama Ensinu

Sesaun Tópiku Objetivu Rezultadu Esperadu Sumáriu
1ª Introdusan no aprezentasaun unidade kurrikulár, no avaliasaun kompeténsia preliminár. Estudante sira komprende estrutura programa durante semestre ne'e nian no identifika kompeténsia inisiál. Estudante koñese programa, regra no espetativa, no dosente iha vizaun kona-ba estudante sira-nia kompeténsia. 🔗
2ª Introdusaun ba intelijénsia artifisiál. Estudante sira komprende konseitu báziku, evolusaun, no aplikasaun IA nian. Estudante sira bele esplika konseitu báziku IA, evolusaun, no área aplikasaun prinsipál sira. 🔗
3ª Ajente intelijente (intelligent agents). Estudante sira komprende konseitu ajente, ambiente, no interasaun entre sira. Estudante sira bele esplika estrutura no komportamentu ajente intelijente ida. 🔗
4ª Algoritmia peskiza(search). Estudante sira komprende problema peskiza no algoritmu klásiku sira hanesan breadth-first search (BFS), depth-first search (DFS), no A* search. Estudante sira bele aplika algoritmu BFS, DFS, no A* atu rezolve problema peskiza simples. 🔗
5ª Fundamentu machine learning (ML) no avaliasaun. Estudante sira komprende konseitu báziku ML, pipeline, tipu aprendizajen, no métodu avaliasaun modelu. Estudante sira bele deskreve pipeline ML, diferensia entre supervised, unsupervised no reinforcement learning, no avalia modelu simples ida. 🔗
6ª Aprendizajen supervizionada (supervised learning). Estudante sira komprende lalaok aprendizajen supervizionada, inklui mapeamentu hosi input ba output etiketadu, diferensa entre klasifikasaun no regresaun, no forma avalia modelu. Estudante sira bele esplika aprendizajen supervizionada iha sira-nia liafuan rasik, distinge klasifikasaun no regresaun, deskreve oinsá modelu aprende hosi dadus etiketadu, no interpreta rezultadu avaliasaun sira. 🔗
7ª Aula prátika: Treinu no avaliasaun modelu klasifikasaun. Estudante sira aplika aprendizajen supervizionada uza scikit-learn atu treinu, avalia, no kompara modelu simples sira. Estudante sira bele load no prepara dataset, halo separasaun dadus ba treinu no teste, treinu pelumenus modelu rua, avalia ninia performance, no kompara no esplika modelu ida-ne'ebé mak di'ak liu. 🔗
8ª Aprendizajen la'ós supervizionada (usupervised learning). 🔗
9ª Aula prátika: Treinu no avalia modelu clustering. 🔗
10ª Reinforcement learning. 🔗
11ª Introdusaun ba deep learning. 🔗
12ª Backpropagation, loss function, no avaliasaun. 🔗
13ª Aula prátika: kria modelu IA ida uza téknika deep learning. 🔗
14ª Deep learning ba vizaun komputasionál. 🔗
15ª Aula prátika: Treinu modelu ida ba klasifikasaun imajen. 🔗
16ª Introdusaun ba transfer learning. 🔗
17ª Ezame médiu semestre 🔗
18ª Aula prátika: fine-tuning modelu pre-treinadu ida ba tarefa klasifikasaun. 🔗
19ª Fundamentu ba NLP no reprezentasaun testu. 🔗
20ª Aula prátika: Fine-tuning word embedings no kria klasifikadór ida ba testu. 🔗
21ª Deep Learning ba NLP no Transformers. 🔗
22ª Aula prátika: Kria klasifikadór ida ba testu uza RNNs no uza modelu pre-treinadu ida iha Hugging Face. 🔗
23ª IA jenerativu. 🔗
24ª Aula prátika: prompt engineering ba LLMs. 🔗
25ª Retrieval-augmented generations no ajente IA. 🔗
26ª Aula prátika: kria ajente IA ida ba atendimentu. 🔗
27ª Introdusaun ba Streamlit Framework. 🔗
28ª Aula prátika: kria protótipu IA ida uza Streamlit. 🔗
29ª Introdusaun ba étika, privasidade, no impaktu sosiál hosi IA. 🔗
30ª Ezame finál 🔗
